jQuery FAQs

Apa itu JavaScript?

Javascript adalah salah satu bahasa pemograman yang bisa digunakan untuk membuat web application, aplikasi di server, aplikasi di PC, Tablet, Smartphone hingga ke microcontroller.

Apa itu jQuery?

jQuery adalah sebuah Javascript library yang menyediakan banyak sekali method yang bisa kita gunakan untuk mempermudah pemograman Javascript.

Apa itu DOM?

Sebuah bahasa netral yang dapat yang mengizinkan javascript atau script lainya mampu mengakses dan mengubah contents, structures dan style dari sebuah document.
